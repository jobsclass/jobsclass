# 문제 해결 중심 카테고리 시스템

## 🎯 핵심 철학
"고객은 무엇을 사고 싶어하는가?"가 아니라  
"고객은 어떤 문제를 해결하고 싶어하는가?"

---

## 📋 카테고리 구조

### 1. 💰 **수익 창출** (Monetization)
**문제**: "돈을 더 벌고 싶어요"
- 부업/사이드잡 시작하기
- 프리랜싱으로 전환하기
- 온라인 비즈니스 시작하기
- 수익 다각화 전략

**예시 서비스**:
- "월 100만원 부업 수익 만들기"
- "프리랜서 첫 고객 받는 법"
- "N잡러 로드맵 컨설팅"

---

### 2. 🚀 **비즈니스 성장** (Business Growth)
**문제**: "내 사업을 키우고 싶어요"
- 온라인 확장 전략
- 마케팅/브랜딩
- 고객 확보 방법
- 매출 증대 전략

**예시 서비스**:
- "소상공인 온라인 진출 가이드"
- "인스타그램 마케팅 완전정복"
- "월 매출 3배 늘리는 법"

---

### 3. ⏰ **시간 자유** (Time Freedom)
**문제**: "시간과 장소에 구애받지 않고 일하고 싶어요"
- 재택근무/원격근무
- 자동화 시스템 구축
- 효율적인 업무 관리
- 워라밸 실현

**예시 서비스**:
- "디지털 노마드 되는 법"
- "자동 수익 시스템 만들기"
- "하루 4시간만 일하기"

---

### 4. 🎯 **전문성 활용** (Expertise Monetization)
**문제**: "내 전문성을 돈으로 바꾸고 싶어요"
- 강사/멘토 전환
- 컨설팅 시작하기
- 온라인 강의 제작
- 전자책/콘텐츠 판매

**예시 서비스**:
- "개발자의 온라인 강의 만들기"
- "디자이너의 템플릿 판매 전략"
- "1:1 코칭 비즈니스 시작"

---

### 5. 💼 **커리어 전환** (Career Transition)
**문제**: "새로운 분야로 이직하고 싶어요"
- 직무 전환 준비
- 포트폴리오 제작
- 이력서/면접 전략
- 취업/이직 컨설팅

**예시 서비스**:
- "비전공자 개발자 되기"
- "마케터로 전직하는 법"
- "6개월 만에 UX 디자이너"

---

### 6. 🎨 **창작/제작** (Creation)
**문제**: "내 작품/콘텐츠를 만들고 싶어요"
- 웹사이트/앱 개발
- 디자인/브랜딩
- 영상/콘텐츠 제작
- 글쓰기/출판

**예시 서비스**:
- "웹사이트 1:1 제작 대행"
- "브랜드 로고 디자인"
- "유튜브 영상 편집 클래스"

---

### 7. 📚 **스킬 습득** (Skill Development)
**문제**: "새로운 기술을 배우고 싶어요"
- 프로그래밍 교육
- 디자인 툴 마스터
- 마케팅 스킬
- 언어/자격증

**예시 서비스**:
- "코딩 부트캠프"
- "피그마 완전정복"
- "구글 애즈 마스터 클래스"

---

### 8. 🏢 **조직/팀 관리** (Team Management)
**문제**: "팀을 잘 이끌고 싶어요"
- 리더십 개발
- 조직 문화 구축
- 채용/인사 전략
- 프로젝트 관리

**예시 서비스**:
- "스타트업 팀 빌딩 컨설팅"
- "신입 개발자 온보딩 전략"
- "애자일 코칭"

---

## 🎨 UI 구성 (참고: 잡스클래스)

### 검색창
```
"어떤 서비스가 필요하신가요? (예: 비즈니스 브랜딩, 마케팅 전략)"
```

### 카테고리 필터 버튼
```
[전체] [💰 수익창출] [🚀 비즈니스성장] [⏰ 시간자유] 
[🎯 전문성활용] [💼 커리어전환] [🎨 창작/제작] [📚 스킬습득] [🏢 조직관리]
```

---

## 💡 차별화 포인트

### 기존 방식 (분야별)
- "IT개발" → 너무 광범위, 내가 뭘 원하는지 모름
- "디자인" → 로고? 웹? 브랜드?

### 문제 중심 (JobsBuild)
- "💰 수익 창출" → 명확한 니즈, 바로 액션
- "🚀 비즈니스 성장" → 구체적인 문제, 해결책 제시

---

## 📊 데이터베이스 필드

```typescript
// websites 테이블
{
  problem_category: 'monetization' | 'business-growth' | 'time-freedom' | 
                    'expertise' | 'career' | 'creation' | 'skill' | 'management',
  problem_statement: '월 100만원 부업 수익 만들기', // 고객의 문제
  solution_approach: '5단계 부업 로드맵 제공', // 해결 방법
}
```
