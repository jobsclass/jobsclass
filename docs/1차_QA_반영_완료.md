**테이블**
- 파트너 정보, 인증 상태, 활동 상태
- 크레딧, 매출, 가입일
- 관리 버튼 (승인/거절/활동 중지)

#### **서비스 관리** (`/admin/services`)
**통계**
- 전체 서비스, 게시됨, 초안, 보관됨

**필터**
- 제목/파트너 이름 검색
- 상태 필터 (게시됨/초안/보관됨)
- 타입 필터 (6가지 서비스 타입)

**테이블**
- 서비스 정보 (썸네일, 제목, 카테고리)
- 파트너 정보, 타입, 가격, 상태
- 관리 버튼 (서비스 보기/게시/보관)

#### **푸터 설정** (`/admin/settings/footer`)
**관리 항목**
- 회사명
- 사업자등록번호
- 통신판매업신고번호
- 사업장 주소
- 대표 이메일
- 대표 전화번호
- 운영시간

**기능**
- 실시간 미리보기
- 저장 버튼
- 새로고침 버튼

#### **DB 마이그레이션**
**site_settings 테이블**
```sql
- id (UUID PK)

- value (JSONB)
- category (TEXT: footer/main_content/hero/pricing/general)
- description (TEXT)
- updated_by (UUID FK)
- created_at, updated_at (TIMESTAMPTZ)
```
