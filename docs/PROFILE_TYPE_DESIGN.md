# 🏢 잡스빌드 - 개인/조직 웹사이트 구조 설계

## 📋 핵심 컨셉

### 🎯 목표
- **개인**: 프리랜서, 전문가, 크리에이터 → 개인 브랜드 웹페이지
- **조직**: 스타트업, 중소기업, 에이전시 → 회사 소개 웹사이트

## 📦 서비스/제품 등록 타입

### 2️⃣ **서비스 등록 시 3가지 타입 선택**

#### 🛒 **Type A: 바로 결제 (직접 판매)**
- **용도**: 온라인 강의, 전자책, 템플릿, 컨설팅
- **기능**: 
  - 가격 설정
  - 결제 버튼
  - 주문 관리
  - 자동 배송 (디지털 상품)
- **예시**: "React 강의 ₩99,000 - 지금 구매하기"

#### 🔗 **Type B: 외부 링크 (서비스 소개)**
- **용도**: 별도 도메인 서비스, 외부 플랫폼, 파트너십
- **기능**:
  - 서비스 소개
  - 외부 링크 버튼 ("서비스 바로가기", "자세히 보기")
  - 새 탭에서 열림
  - 링크 클릭 통계
- **예시**: 
  - "잡스빌드 → https://jobsbuild.com"
  - "잡스벤처스 → https://jobsventures.com"
  - "잡스마켓 → https://jobsmarket.com"

#### 💬 **Type C: 문의 받기 (Lead Generation)**
- **용도**: 맞춤형 서비스, 고가 컨설팅, B2B
- **기능**:
  - 서비스 소개
  - "문의하기" 버튼
  - 문의 폼 팝업
  - 이메일 알림
  - 고객 DB 관리
- **예시**: "기업 마케팅 컨설팅 - 견적 문의"

## 🗂️ 데이터베이스 스키마 (수정)

### 4️⃣ **services 테이블 확장**

```sql
ALTER TABLE services
ADD COLUMN service_type TEXT NOT NULL DEFAULT 'direct_sale' 
  CHECK (service_type IN ('direct_sale', 'external_link', 'inquiry'));

-- Type B용: 외부 링크
ADD COLUMN external_url TEXT;
ADD COLUMN external_button_text TEXT DEFAULT '서비스 바로가기';

-- Type C용: 문의 설정
ADD COLUMN inquiry_enabled BOOLEAN DEFAULT FALSE;
ADD COLUMN inquiry_description TEXT;
```

### 5️⃣ **user_profiles 테이블 확장**

```sql
ALTER TABLE user_profiles
ADD COLUMN profile_type TEXT NOT NULL DEFAULT 'individual'
  CHECK (profile_type IN ('individual', 'organization'));

-- 조직 정보 (profile_type = 'organization' 일 때)
ADD COLUMN organization_name TEXT; -- 조직명 (예: (주)잡스클라스)
ADD COLUMN organization_description TEXT; -- 회사 소개
ADD COLUMN founded_year INTEGER; -- 설립년도
ADD COLUMN team_size TEXT; -- 팀 규모 (1-10명, 11-50명, etc.)
ADD COLUMN organization_links JSONB; -- 소셜/링크
```

## 📱 화면 구성

### 6️⃣ **온보딩 플로우 (수정)**

#### Step 1: 프로필 타입 선택
```
질문: "어떤 웹사이트를 만들고 싶으신가요?"

[ ] 👤 개인 웹페이지
    프리랜서, 전문가, 크리에이터

[ ] 🏢 조직 웹사이트
    스타트업, 회사, 에이전시
```

#### Step 2-A: 개인 (기존 유지)
```
- 무슨 일을 하시나요?
- 주로 누구를 도와주시나요?
- 어떤 서비스를 제공하나요?
- 경력이 어떻게 되시나요?
- 특별한 성과가 있나요?
```

#### Step 2-B: 조직 (신규)
```
- 회사/조직 이름은?
- 어떤 서비스를 제공하나요?
- 주요 고객은 누구인가요?
- 회사 소개 (한 줄)
- 주요 제품/서비스는? (3개)
```

### 7️⃣ **서비스 등록 플로우 (수정)**

#### Step 1: 서비스 타입 선택
```
┌─────────────────────────────────────┐
│  어떤 형태로 제공하시나요?          │
├─────────────────────────────────────┤
│  🛒 바로 결제                        │
│  고객이 즉시 구매 가능              │
│  (온라인 강의, 전자책 등)           │
│                                     │
│  🔗 외부 링크                        │
│  별도 웹사이트로 연결               │
│  (독립 서비스, 파트너십)            │
│                                     │
│  💬 문의 받기                        │
│  견적 상담 후 진행                  │
│  (맞춤형 컨설팅, B2B)               │
└─────────────────────────────────────┘
```

#### Step 2: 타입별 입력 폼

**🛒 바로 결제**:
- 서비스명, 설명, 가격, 특징, 썸네일

**🔗 외부 링크**:
- 서비스명, 설명, 외부 URL, 버튼 텍스트, 썸네일

**💬 문의 받기**:
- 서비스명, 설명, 문의 안내 문구, 썸네일
