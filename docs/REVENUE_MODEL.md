## 📊 AI 기능 비용 분석 (OpenAI GPT-4o-mini + DALL-E 3)

### 1️⃣ 온보딩 (웹사이트 자동 생성)
**사용 모델**: GPT-4o-mini
- 입력 토큰: ~500 토큰 (사용자 답변 5개)
- 출력 토큰: ~3,000 토큰 (프로필 + 서비스 3개 + 블로그 5개 + 포트폴리오 3개 + 커리어)
- **비용**: $0.00015 (입력) + $0.0006 (출력) = **$0.00075** (약 ₩1원)

### 2️⃣ AI 이미지 생성 (DALL-E 3)
**사용 모델**: DALL-E 3 (1024x1024, standard quality)
- 서비스 이미지 3개: $0.04 × 3 = $0.12 (약 ₩160)
- 블로그 이미지 (선택): $0.04 × 5 = $0.20 (약 ₩265)
- **최소 비용**: $0.12 (서비스만) ~ **최대 $0.32** (약 ₩425)

### 3️⃣ 추가 AI 카피라이팅 (서비스/블로그 수정)
**사용 모델**: GPT-4o-mini
- 1회 수정당: ~500 토큰 (입력) + ~300 토큰 (출력)
- **비용**: $0.0001 ~ $0.0002 (약 ₩0.1~0.3원)
