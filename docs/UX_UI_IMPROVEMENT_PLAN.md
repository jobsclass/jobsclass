## 📊 현재 문제점 분석

### 1. 마켓플레이스 필터 (🔴 높은 우선순위)
**문제:**
- 필터 섹션이 너무 커서 스크롤이 필요함
- 첫 화면에서 서비스 목록이 안 보임
- 사용자가 즉시 서비스를 볼 수 없음

**해결책:**
- 필터를 컴팩트하게 재설계
- 드롭다운/토글 방식으로 변경
- Sticky 필터 (스크롤해도 상단 고정)
- 서비스 목록을 우선 표시

### 2. Footer 메뉴 (🟡 중간 우선순위)
**문제:**
- 페이지가 없는 링크들이 많음
- "채용" 메뉴 불필요
- 클릭하면 404 또는 빈 페이지

**해결책:**
- 실제 존재하는 페이지만 노출
- 채용 메뉴 제거
- 필수 링크만 유지:
  - 마켓플레이스
  - 이용약관 (추후 작성)
  - 개인정보처리방침 (추후 작성)

### 3. 페이지 출구 (🔴 높은 우선순위)
**문제:**
- 대부분의 페이지에 뒤로가기 버튼 없음
- 사용자가 막다른 골목에 갇힘
- 특히 대시보드 내부 페이지들

**해결책:**
- 모든 상세/편집 페이지에 "뒤로가기" 추가
- 대시보드 → 서브페이지 → 뒤로가기 버튼
- Breadcrumb 네비게이션 추가

### 4. 로그인 상태별 네비게이션 (🔴 높은 우선순위)
**문제:**
- 비로그인 vs 로그인 상태 구분 불명확
- 파트너 vs 클라이언트 메뉴 구분 없음

**해결책:**
- 비로그인: 홈, 마켓플레이스, 로그인, 회원가입
- 파트너: 홈, 마켓플레이스, 내 서비스, 대시보드, 프로필
- 클라이언트: 홈, 마켓플레이스, 내 구매, 크레딧, 프로필

### 2. 마켓플레이스 (/marketplace)
**현재 상태:** ⚠️ 개선 필요

**문제점:**
- 필터 섹션이 너무 큼 (높이 800px 이상)
- 첫 화면에서 서비스 목록 안 보임
- 필터 항목이 세로로 너무 길게 나열됨

**개선 계획:**
```
Before:
┌─────────────────┐
│ GNB             │
├─────────────────┤
│ 검색창          │  ← 여기서 이미 100px
├─────────────────┤
│ 카테고리 필터   │  ← 200px
│ (8개 버튼)      │
├─────────────────┤
│ 서비스 타입     │  ← 300px
│ (10개 체크박스) │
├─────────────────┤
│ 가격 범위       │  ← 100px
├─────────────────┤
│ 정렬            │  ← 100px
└─────────────────┘
  ← 여기서 스크롤해야 서비스 보임!

After:
┌─────────────────────────────────┐
│ GNB                             │
├─────────────────────────────────┤
│ 검색창 + 필터 버튼 (한 줄)     │ ← 60px
├─────────────────────────────────┤
│ 카테고리 탭 (가로 스크롤)      │ ← 80px
├─────────────────────────────────┤
│ 서비스 목록 시작! ✅           │
│                                 │
│ [서비스 카드]                   │
│ [서비스 카드]                   │
└─────────────────────────────────┘
```

**구체적 변경:**
1. 필터를 사이드바 또는 모달로 변경
2. 카테고리를 가로 탭으로 변경
3. 서비스 타입을 드롭다운으로 축소
4. 첫 화면에서 바로 서비스 보이게

## 🎨 디자인 시스템 표준화

### 버튼 위계
```typescript
// Primary (주요 액션)
bg-gradient-to-r from-primary-500 to-purple-500

// Secondary (부가 액션)
bg-white/10 border border-white/20

// Danger (삭제 등)
bg-red-500 hover:bg-red-600

// Back (뒤로가기)
bg-gray-800 hover:bg-gray-700
```

### 간격 표준
```typescript
// 섹션 간격
py-20 (모바일: py-12)

// 카드 간격
gap-6

// 컨테이너
container mx-auto px-4 max-w-7xl
```
